# Multiplayer Dice Game

## Description
A browser-based multiplayer dice game built using HTML, CSS, and JavaScript.
The game supports multiple players, turn-based logic, and a timer-based skip system.

## Features
- Supports multiple players
- Turn-based dice rolling
- Timer-based turn skipping
- Score tracking with winning condition

## Tech Stack
HTML, CSS, JavaScript

## What I Learned
- JavaScript DOM manipulation
- Event-driven programming
- Managing application state
- Working with timers and localStorage
